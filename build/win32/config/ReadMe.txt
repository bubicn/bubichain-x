//Configuration file version description.

The execution of the program must depend on the configuration file

//mainnet config
config-mainnet.json 			

//testnet config
config-testnet.json			

//single config
config-single.json			


Please copy one of these files to bubi.json first! if not, it can not run

like this:
	cp config-mainnet.json config.json
or
	cp config-testnet.json config.json
or
	cp config-single.json config.json
